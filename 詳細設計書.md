## 1. API設計（各エンドポイントのリクエスト・レスポンスの詳細）
## 1.1. 各メソッドの機能

| メソッド | パス | 機能 |
|---|---|---|
| POST | `/inquiries` | 問い合わせ登録・保存 |
| GET | `/inquiries` | 問い合わせ一覧を取得する |
| GET | `/inquiries/{id}` | 問い合わせ詳細取得 |

### 1.2. POST /inquiries 問い合わせ登録
### 概要： 問い合わせ内容を受け取り、Gemini AI によるカテゴリー分類・緊急度判定・回答案生成を行い、JSONファイルへ保存する。
```http
POST http://localhost:8000/inquiries
Content-Type: application/json
```

```json
{
  "question": "有給休暇の申請方法を教えてください。"
}
```

### 各フィールドの型・説明

| フィールド名 | 型           | 説明          | 必須 |
|---|---|---|---|
| question     | string       | 問い合わせ内容 （空文字不可） |  Yes  |

### 成功レスポンス（201）

```json
[
  {
    "id": 1,    #連番整数ID
    "created_at": "2026-05-15T10:00:00+09:00",
    "question": "有給休暇の申請方法を教えてください。",
    "category": "休暇",
    "urgency": "低",
    "answer": "有給休暇の申請は..."
  }
]
```

### エラーレスポンス

|ステータス|発生条件|
|-|-|
|400|question が未入力|
|503|Gemini API 接続失敗|

## 3 JSON 読み書き設計
ファイル：`backend/storage.py`

### 3.1. 全件読み込み（load_inquiries）
```text
1. data/inquiries.json のファイルが存在するか確認する
2. 存在しない場合は空のリスト [] を返す
3. 存在する場合は json.load() で読み込む
4. リストを返す
```

### 3.2. 1件追記保存（save_inquiry）
```text
1. 全件読み込みを行いリストを取得する
2. 新しいレコードをリストの末尾に append する
3. json.dump() でファイルに上書き保存する（ensure_ascii=False, indent=2）
```

### 3.3. ID検索（find_by_id）
```text
1. 全件読み込みを行いリストを取得する
2. id が一致するレコードを検索する
3. 見つかれば返す。見つからなければ None を返す
```

### 5.2. 問い合わせ入力画面 
```python
if page == "AI問い合わせ入力":
    st.title("AI問い合わせアプリ")

    with st.form("inquiry_form"):
        question = st.text_area("問い合わせ入力", height=100)
        submitted = st.form_submit_button("送信")

    if submitted:
        try:
            # 未入力チェック
            if question.strip() == "":
                st.error("問い合わせ内容を入力してください。")

            else:
                with st.spinner("回答を生成中..."):
                    time.sleep(2)

                    response = requests.post(
                        API_URL,
                        json={
                            "question": question
                        },
                        timeout=50
                    )

                # ステータス確認
                if response.ok:
                    result = response.json()

                    st.success("問い合わせを登録しました。")

                    st.write("カテゴリー：", result["category"])
                    st.write("緊急度：", result["urgency"])
                    st.write("回答案：", result["answer"])

                else:
                    st.error("登録に失敗しました。")

        except requests.exceptions.RequestException as e:
            st.error(f"APIへのリクエストに失敗しました: {e}")
```
