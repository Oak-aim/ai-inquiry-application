## 1.2. POST /inquiries 問い合わせ登録
### 概要

問い合わせ内容を受け取り、Gemini AI によるカテゴリー分類・緊急度判定・回答案生成を行い、JSONファイルへ保存する。

```http
POST http://localhost:8000/inquiries
Content-Type: application/json
```

### リクエスト

```json
{
  "question": "有給休暇の申請方法を教えてください。"
}
```

### 各フィールドの型・説明

| フィールド名   | 型      | 説明             | 必須  |
| -------- | ------ | -------------- | --- |
| question | string | 問い合わせ内容（空文字不可） | Yes |

### 成功レスポンス（201）

```json
{
  "id": 1,
  "created_at": "2026-05-15T10:00:00+09:00",
  "question": "有給休暇の申請方法を教えてください。",
  "category": "休暇",
  "urgency": "低",
  "answer": "有給休暇の申請は..."
}
```

### エラーレスポンス

| ステータス | 発生条件            |
| ----- | --------------- |
| 400   | question が未入力   |
| 500   | Geminiレスポンス解析失敗 |
| 503   | Gemini API 接続失敗 |

# 2. データモデル（Pydantic）

ファイル：`backend/main.py`

```python
from pydantic import BaseModel
from datetime import datetime

class InquiryRequest(BaseModel):
    question: str


class InquiryRecord(BaseModel):
    id: int
    created_at: datetime
    question: str
    category: str
    urgency: str
    answer: str
```

### APIレスポンスモデル

```python
@app.post("/inquiries", response_model=InquiryRecord)
@app.get("/inquiries", response_model=list[InquiryRecord])
@app.get("/inquiries/{id}", response_model=InquiryRecord)
```

| クラス            | 用途                         |
| -------------- | -------------------------- |
| InquiryRequest | フロントエンドから受け取るPOSTリクエストのボディ |
| InquiryRecord  | JSONファイルへ保存・APIから返却するデータ   |

# 3. JSON 読み書き設計

ファイル：`backend/storage.py`

## 3.1. 全件読み込み（_load）

```text
1. data/inquiries.json のファイルが存在するか確認する
2. 存在しない場合は空のリスト [] を返す
3. 存在する場合は json.load() で読み込む
4. リストを返す
```

---

## 3.2. 1件追記保存（save_inquiry）

```text
1. data ディレクトリが存在するか確認する
2. 存在しない場合は作成する
3. 全件読み込みを行いリストを取得する
4. 新しいレコードをリストの末尾に append する
5. json.dump() でファイルへ上書き保存する
6. 保存時は ensure_ascii=False, indent=2 を指定する
```

---

## 3.3. ID検索（get_inquiry_by_id）

```text
1. 全件読み込みを行いリストを取得する
2. id が一致するレコードを検索する
3. 見つかれば返す
4. 見つからなければ None を返す
```

## 5.3. 問い合わせ入力画面

```python
import time

if page == "問い合わせ入力":

    st.title("問い合わせアプリ")

    with st.form("inquiry_form"):

        question = st.text_area(
            "問い合わせ入力",
            height=100
        )

        submitted = st.form_submit_button("送信")

    if submitted:

        try:

            # 未入力チェック
            if question.strip() == "":

                st.error(
                    "問い合わせ内容を入力してください。"
                )

            else:

                with st.spinner("回答を生成中..."):

                    time.sleep(2)

                    response = requests.post(
                        f"{API_URL}/inquiries",
                        json={
                            "question": question
                        },
                        timeout=50
                    )

                # ステータス確認
                if response.status_code == 201:

                    result = response.json()

                    st.success(
                        "問い合わせを登録しました。"
                    )

                    st.write(
                        "カテゴリー：",
                        result["category"]
                    )

                    st.write(
                        "緊急度：",
                        result["urgency"]
                    )

                    st.write(
                        "回答案：",
                        result["answer"]
                    )

                elif response.status_code == 400:

                    st.error(
                        "問い合わせ内容を入力してください。"
                    )

                elif response.status_code == 503:

                    st.error(
                        "AI処理に失敗しました。"
                    )

                elif response.status_code == 500:

                    st.error(
                        "AIレスポンス解析に失敗しました。"
                    )

                else:

                    st.error("登録に失敗しました。")

        except requests.exceptions.RequestException as e:

            st.error(
                f"APIへのリクエストに失敗しました: {e}"
            )
```

## 5.4. 一覧画面

```python
if page == "一覧":

    st.title("問い合わせ一覧")

    response = requests.get(
        f"{API_URL}/inquiries"
    )

    if response.status_code == 200:

        records = response.json()

        if len(records) == 0:

            st.info(
                "問い合わせ履歴がありません。"
            )

        else:

            # ヘッダー行
            col1, col2, col3, col4, col5 = st.columns(
                [1, 3, 4, 2, 2]
            )

            col1.write("ID")
            col2.write("登録日時")
            col3.write("問い合わせ内容")
            col4.write("カテゴリー")
            col5.write("詳細")

            # データ行
            for record in records:

                col1, col2, col3, col4, col5 = st.columns(
                    [1, 3, 4, 2, 2]
                )

                col1.write(record["id"])
                col2.write(record["created_at"])
                col3.write(record["question"])
                col4.write(record["category"])

                if col5.button(
                    "詳細",
                    key=f'detail_{record["id"]}'
                ):

                    st.session_state["selected_id"] = record["id"]
                    st.session_state["page"] = "詳細"

                    st.rerun()

    else:

        st.error("一覧取得に失敗しました。")
```

## 5.5. 詳細画面

if st.session_state.get("page") == "詳細":

    st.title("問い合わせ詳細")

| 5  | FastAPI（Geminiレスポンス） | JSON形式が不正             | 500 を返す     | 「AIレスポンス解析に失敗しました。」   |
