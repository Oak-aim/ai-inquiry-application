# 詳細設計書
### プロジェクト名：AI Inquiry Application 
### 作成日：2026年5月15日 

## 1. API設計（各エンドポイントのリクエスト・レスポンスの詳細）
## 1.1. 各メソッドの機能

| メソッド | パス | 機能 |
|---|---|---|
| POST | `/inquiries` | 問い合わせ登録・保存 |
| GET | `/inquiries` | 問い合わせ一覧を取得する |
| GET | `/inquiries/{id}` | 問い合わせ詳細取得 |

### 1.2. POST /inquiries 問い合わせ登録（タスク情報を受け取り、JSON ファイルに保存する）

```
HTTPメソッド  --> POST
URL          --> http://localhost:8000/tasks
Content-Type --> application/json
```

```json
[
  {
    "content": "買い物する",
    "category": "プライベート",
    "priority": "低"
  }
]
```

### 各フィールドの型・説明

| フィールド名 | 型           | 説明          | 必須 |
|---|---|---|---|
| content     | string       | タスク内容 （空不可） |  Yes  |
| category    | string       | 仕事 / プライベート / その他  |  Yes  |
| priority    | string       | 高 / 中 / 低 |  Yes  |

### 成功レスポンス（成功 201）

```json
[
  {
    "id": 1,    #連番整数ID
    "created_at": "2026-05-13T10:00:00+09:00",
    "content": "買い物する",
    "category": "プライベート",
    "priority": "低",
    "status": "未完了"
  }
]
```

---

### 1.3. GET /tasks タスク一覧取得（保存済みのタスクを全件、登録日時の降順で返す）

```
HTTPメソッド  --> GET
URL          --> http://localhost:8000/tasks
```

### 仕様

- 登録日時（created_at）の降順で返す
- 件数が0件のときは空のリスト [ ] を返す

### 成功レスポンス（成功 200）

```json
[
  {
    "id": 2,    #連番整数ID
    "created_at": "2026-05-13T10:00:00+09:00",
    "content": "買い物する",
    "category": "プライベート",
    "priority": "低",
    "status": "未完了"
  },
  {
    "id": 1,    #連番整数ID
    "created_at": "2026-05-13T10:00:00+09:00",
    "content": "エサをやる",
    "category": "その他",
    "priority": "中",
    "status": "未完了"
  }
]
```

### 1.4. GET /tasks/{id} タスク詳細取得（指定したIDのタスクを1件返す）

```
HTTPメソッド  --> GET
URL          --> http://localhost:8000/tasks/{id}
```

### 成功レスポンス（成功 200）

```json
[
  {
    "id": 1,    #連番整数ID
    "created_at": "2026-05-13T10:00:00+09:00",
    "content": "買い物する",
    "category": "プライベート",
    "priority": "低",
    "status": "未完了"
  }
]
```

### 1.5. PATCH /tasks/{id}/complete　タスク完了更新（指定したIDのタスクのステータスを「完了」に変更する、PUT は全体更新だから、使わない）

```
HTTPメソッド  --> PATCH
URL          --> http://localhost:8000/tasks/{id}/complete
```

### 成功レスポンス（成功 200）

```json
[
  {
    "id": 1,    #連番整数ID
    "created_at": "2026-05-13T10:00:00+09:00",
    "content": "買い物する",
    "category": "プライベート",
    "priority": "低",
    "status": "完了"
  }
]
```

### エラーレスポンス

| ステータス | 発生条件 | 意味 |
|---|---|---|
| 404 | ページ・id・APIが存在しない | Not Found |
| 503 | API 接続失敗 | Service Unavailable |
