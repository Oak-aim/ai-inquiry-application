## 4. 処理フロー
### 4.1. 問い合わせ入力フロー
```
[ユーザー] 問い合わせ内容を入力する
        ↓
[Streamlit] テキストエリアへ入力内容を保持する
        ↓
[ユーザー] 「送信」ボタンを押す
        ↓
[Streamlit] 入力内容が空でないかチェックする
        ↓
[Streamlit] 未入力の場合はエラーメッセージを表示して処理を停止する
        ↓
[Streamlit] Fast API へ問い合わせ内容(POST/inquiries)で送信する
```

### 4.2. AIカテゴリー・緊急度・回答案生成フロー
```
[FastAPI] 問い合わせ内容を Gemini AI へ送信する
        ↓
[Gemini AI] カテゴリー・緊急度・回答案を生成する
        ↓
[Gemini AI] 生成結果を FastAPI へ返す
        ↓
[FastAPI] カテゴリー・緊急度・回答案を Streamlit へ返す
        ↓
[Streamlit] カテゴリー・ 緊急度・回答案を問い合わせ入力画面へ表示する   
```

### 4.3. 履歴保存フロー
```
[FastAPI] AI 分析結果を受け取る
        ↓
[FastAPI] 登録時刻・ID を生成する
        ↓
[FastAPI] 問い合わせ内容・カテゴリー・緊急度・回答案を含む問い合わせデータを生成する
        ↓
[FastAPI] data/inquiries.json を読み込む
        ↓
[FastAPI] 問い合わせデータを追加する
        ↓
[FastAPI] data/inquiries.json を保存する
```

### 4.4. 一覧表示フロー
```
[ユーザー] サイドバーで「一覧表示」を選択する
        ↓
[Streamlit] GET /inquiries を FastAPI へ送信する
        ↓
[FastAPI] data/inquiries.json を読み込む
        ↓
[FastAPI] 登録時刻の降順で問い合わせ一覧ソートして返す
        ↓
[Streamlit] 問い合わせ一覧を画面へ表示する
```

### 4.5. 詳細表示フロー
```
[ユーザー] 一覧画面で「詳細」ボタンを押す
        ↓
[Streamlit] 選択した問い合わせIDを保持する
        ↓
[Streamlit] GET /inquiries/{id} を FastAPI へ送信する
        ↓
[FastAPI] data/inquiries.json から該当 ID を検索する
        ↓
[FastAPI] 該当データを Streamlit へ返す
        ↓
[Streamlit] 問い合わせ詳細を表示する
```

## 6. データ設計
### 6.1 JSON構造

```json
[
  {
    "id": 1,
    "created_at": "2026-05-14T10:00:00+09:00",
    "question": "有給休暇の申請方法を教えてください。",
    "category": "休暇",
    "urgency": "低",
    "answer": "有給休暇の申請は..."
  }
]

```

| フィールド名 | 型           | 生成元          |
|---|---|---|
| id          | 整数         | バックエンドで自動採番ID |
| created_at  | 文字列（日時）| バックエンドで自動生成 |
| question    | 文字列       | ユーザー入力     |
| category    | 文字列       | Gemini AI 生成  |
| urgency     | 文字列       | Gemini AI 生成  |
| answer      | 文字列       | Gemini AI 生成  |

---

## 7. エラー処理方針

| エラーの種類           | 発生箇所    | 対処方法                             |
|---|---|---|
|       未入力        | Streamlit  | 未入力エラーメッセージを表示し、送信しないことにする  |
|    API 接続失敗     | FastAPI   | 503 を返す。Streamlit でエラー表示する    |
|  JSONファイルが存在しない  |  FastAPI  | 空のリストとして処理を続行する            |
|   一覧画面・詳細画面・APIが存在しない  | FastAPI |　404 を返し、Streamlit でエラー表示する   |
