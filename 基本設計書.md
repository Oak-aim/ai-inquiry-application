# 基本設計書
### プロジェクト名：AI Inquiry Application 
### 作成日：2026年5月14日 

---

## 1 システム構成
## 1.1 構成図
```
ユーザー（ブラウザ）
  ↓ (localhost:8501)
Streamlit（GUI, frontend/app.py）
  ↓ (localhost:8000)
FastAPI（APIサーバー, backend/main.py）
  ↓
ローカルJSONファイル（データ保存・書き込み）
```

```
ローカルJSONファイル（保存済みデータを読み込む）
  ↓
FastAPI（データ読み込み）
  ↓ (localhost:8000)
Streamlit（画面表示）
  ↓ (localhost:8501)
ユーザー（ブラウザ）
```

## 1.2 フォルダー・ファイル構成
```
プロジェクトルート/
├── frontend/
│   └── app.py                  # Streamlit アプリ
├── backend/
│   ├── main.py                 # FastAPI エントリポイント
│   └── storage.py              # JSON 読み書き
└── data/
    └── tasks.json              # タスク保存ファイル

```
