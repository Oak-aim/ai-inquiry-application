# 基本設計書
### プロジェクト名：AI Inquiry Application 
### 作成日：2026年5月14日 

---

## 1 システム構成
## 1.1 構成図
```
ユーザー（ブラウザ）
  ↓ HTTP (localhost:8501)
Streamlit（GUI）
  ↓ HTTP (localhost:8000)
FastAPI（APIサーバー ）   ➞  Gemini API（Gemini API通信）
  ↓
ローカル JSON ファイル（データ保存）
```

## 1.2 フォルダー・ファイル構成
```
プロジェクトルート/
├── frontend/
│   └── app.py                  # Streamlit アプリ
├── backend/
│   ├── main.py                 # FastAPI エントリポイント
│   └── storage.py              # JSON 読み書き
└── data/
    └── tasks.json              # タスク保存ファイル

```

---

## 2 画面構成とレイアウト
## 2.1 画面構成（画面一覧）
```
タスク登録画面（メイン画面、タスクの作成・登録を行う）
└── タスク一覧画面（登録済みタスクを閲覧する）
      └── タスク詳細画面（詳細確認・完了更新を行う）
```

## 2.2 各画面のレイアウト（Screen`s Layout）
### 問い合せ入力画面（メイン画面）
```
+----------------------------------+
|AI問い合せアプリ                   |
+----------------------------------+
│  [サイドバー]                     │
|  ・問い合せ入力                   |
|  ・一覧表示                       |
|      　     _________________    |
|問い合せ入力 |                |    |
|            |________________|    |
|                                  |
|                        [送信]    | <-- 押下すれば、問い合わせ送信する
|(未入力メッセージ)                 |
| 問い合せ入力してください           | <-- 未入力の場合、エラーメッセージ表示する
| 回答生成中...                     |
|(API 接続不良メッセージ)            |
| エラーメッセージ                   | <-- API 接続不良の場合、エラーメッセージ表示する
|                                  |
| カテゴリ：その他                  |
| 緊急度：中                        |
| 回答案：.....                     |
|                                  |
+----------------------------------+
```
