## 1. システム構成


## 4. API 一覧（FastAPI）

| メソッド | パス | 機能 |
|---|---|---|
| POST | `/inquiries` | 問い合わせ登録・保存 |
| GET | `/inquiries` | 問い合わせ一覧を取得する |
| GET | `/inquiries/{id}` | 問い合わせ詳細取得 |

## 5. データ設計
### 5.1 JSON構造

```json
[
  {
    "id": 1,
    "created_at": "2026-05-14T10:00:00+09:00",
    "question": "有給休暇の申請方法を教えてください。",
    "category": "休暇",
    "urgency": "低",
    "answer": "有給休暇の申請は..."
  }
]

```

| フィールド名 | 型           | 生成元          |
|---|---|---|
| id          | 整数         | バックエンドで自動採番ID |
| created_at  | 文字列（日時）| バックエンドで自動生成 |
| question    | 文字列       | ユーザー入力     |
| category    | 文字列       | Gemini AI 生成  |
| urgency     | 文字列       | Gemini AI 生成  |
| answer      | 文字列       | Gemini AI 生成  |

---

## 6. エラー処理方針

| エラーの種類           | 発生箇所    | 対処方法                            |
|---|---|---|
|       未入力        | Streamlit  | 未入力エラーメッセージを表示し、送信しない   |
|    API 接続失敗     | Fast API   | 503 を返す。Streamlit でエラー表示する    |
| JSONファイルが空 | Fast API    | 空 JSON として処理を続行する       |
|   一覧画面・詳細画面・APIが存在しない  | Fast API | 404 を返す。Streamlit でエラー表示する   |
